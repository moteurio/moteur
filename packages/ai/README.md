# @moteurio/ai

AI layer for Moteur: configurable providers, credits, writing helpers, translation, and generation endpoints composed by `@moteurio/api`.

See [AI](../../docs/AI.md) and the main [documentation index](../../docs/README.md).

## Install

```bash
npm install @moteurio/ai
```

Requires Node **22+** (see `engines` in `package.json`).
