# @moteurio/core

In-process **content engine** for Moteur: projects, models, entries, pages, validation, storage, and plugin integration. Consumed by `@moteurio/api` and tooling; most HTTP integrations should use [`@moteurio/client`](../client/README.md) against the REST API instead.

Documentation: [Developer API (embedded)](../../docs/Developer%20API.md), [main docs index](../../docs/README.md).

## Install

```bash
npm install @moteurio/core
```

Requires Node **22+** (see `engines` in `package.json`).
