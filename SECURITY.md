# Security

If you believe you have found a security vulnerability in Moteur or published `@moteurio/*` packages, please report it responsibly:

- Prefer **private disclosure** (do not open a public issue for unfixed vulnerabilities).
- Contact the maintainers through the security contact published on the [GitHub repository](https://github.com/moteurio/moteur) (Security tab / advisories), or your existing support channel if you have one.

Include steps to reproduce, affected versions or commit, and impact when possible.
