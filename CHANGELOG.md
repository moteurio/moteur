# Changelog

All notable changes to published `@moteurio/*` packages are documented here. Version lines are kept aligned across the monorepo for coordinated releases (see [RELEASING.md](RELEASING.md)).

## 0.1.0

Initial public release line for:

- `@moteurio/types`, `@moteurio/plugin-sdk`, `@moteurio/core`, `@moteurio/ai`, `@moteurio/presence`, `@moteurio/api`, `@moteurio/client`, `@moteurio/cli`

**Semver (0.x):** Until 1.0, minor bumps may include breaking HTTP or npm API changes; patch bumps are fixes. After 1.0, follow [semver](https://semver.org/) for npm packages; HTTP API changes are called out in release notes.
